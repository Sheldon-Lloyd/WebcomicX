Requirements

Before installing WebcomicX make sure your server

  1. Has read-write privilege
  2. Supports asp.net 4.5 or greater

Installation

Before you continue I recommend that you do your first time on a local server before uploading it to a web server

  1. Download the latest version of WebcomicX
  2. Extract the zip file
  3. Choose the desired location of the folder
  4. Next, you will need to upload the contents of the folder rather than the actual folder to your web server
  5. Once you have uploaded these files to your web server you can navigate to your site and you will be taken through the first time setup process
